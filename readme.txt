Программа для поиска в массиве всеx возможных 3-х ключей, сумма которых дает заданное значение.

Исходные данные подаются через стандартный ввод. Результат работы программы должен печаться на экране.
Для тестирования программы локально воспользуйтесь скриптом run_test.py.
Пример запуска: python run_test.py ./task1.exe ./input.txt
Результат работы должен быть аналогичен output.txt, т.е. содержать ровно 4 строки, каждая из которых содержит ровно 3 пары <ключ>:<строка>. Порядок строк и элементов внутри строки может быть произвольный.

Вход:
0 - заданное значение, т.е. сумма 3-х ключей должна быть равна 0
6 - количество пар: int -> string
1 abs
-1 foo
0 bar
2 x
-3 three
1 again

Ответ (-3 + 1 + 2 = 0, -1 + 0 + 1 = 0):
-3:three 1:abs 2:x
-3:three 1:again 2:x
-1:foo 0:bar 1:abs
-1:foo 0:bar 1:again
